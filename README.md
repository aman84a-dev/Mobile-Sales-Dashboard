🚀 Project Overview

This project is an interactive Mobile Sales Dashboard built using Power BI to analyze sales performance and customer insights.

The dashboard helps track key metrics, identify trends, and support data-driven decision-making.

---

📌 Key Features

- 📊 Total Sales Analysis
- 📦 Quantity & Transactions Tracking
- 💰 Average Price Monitoring
- 🌍 City-wise Sales Visualization (Map)
- 📈 Monthly Sales Trend Analysis
- ⭐ Customer Rating Insights
- 💳 Payment Method Distribution
- 📅 MTD (Month-to-Date) Reporting
- 🔄 Same Period Last Year Comparison

---

🛠 Tools & Technologies Used

- 📊 Power BI
- 🧠 DAX
- 🔄 Power Query
- 📁 Excel / CSV Dataset
